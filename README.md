# SmartStay-AI

Week 2 frontend skeleton built with React, Tailwind CSS, and React Router.

## Getting Started

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Pages

| Route        | Page      |
| ------------ | --------- |
| `/`          | Home      |
| `/about`     | About     |
| `/dashboard` | Dashboard |
| `/login`     | Login     |
